:even, :odd匹配结果集中顺序为偶数或奇数的元素
:header匹配标题元素H1、H2、H3等标签
:not不匹配后面选择器的元素
:eq(index)匹配顺序号等于index的元素
:gt(index)匹配顺序号大于index的元素
:lt(index)匹配顺序号小于index的元素
:first-child匹配第一个子元素
:last-child匹配最后一个子元素
:only-child匹配唯一的子元素
:nth-child(index)匹配第index个子元素
:has(p)匹配包含p元素的元素
:contains('some text')匹配包含some text文本的元素
:empty匹配无内容的元素
:parent匹配拥有子元素的元素
:hidden匹配不可见的元素
:visible匹配可见的元素
:animated匹配正处于动画过程中的元素
 
$('[attribute*=value]')attribute like %value%'
$('[attribute|=value]')attribute=value或attribute like value-%
$('[attribute~=value]')value由多个子串组成以空格分隔, attribute包含任意一个子串
$('[attribute^=value]')attribute like value%
$('[attribute=value]')attribute=value
$('[attribute$=value]')attribute like %value
$('[attribute!=value]')attribute不等于value
$('[attribute1=value1][attribute2=value2]')同时匹配多个属性及其值
